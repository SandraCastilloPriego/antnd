Add reaction module

Description

This module add a defined reaction in the selected model. The added reaction will be written in the SBML model and also in the information file of the model.

Method parameters

Reaction name
Name of the reaction to be added (it will be the reaction ID) (the reaction name will be added in the next versions of the program.
Compound Ids
Ids of the compounds in the reaction separated by "," (comma).
Stoichiometry
Stoichiometry of each of the compounds in the reaction separated by "," (comma).
Lower bound
Lower bound of the reaction.
Upper bound
Upper bound of the reaction.
