External Fluxes Visualization

Description

This module creates a new SBML model from the fluxes calculated with any external program such a sybil (in R) or cobra (in matlab). The output SBML file also contains the graph to be visualized. It can be done by selecting the new model in the left space, clicking the right button of the mouse, and selecting the option "Visualize".

The input files format should be csv (comma separated file) with two columns: "Reaction ID" and "Fluxes". The name in the header is not relevant. The important thing is the order of the columns.
